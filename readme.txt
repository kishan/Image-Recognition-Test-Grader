This term project is meant to serve as a tool for teachers to make the process of gaming multiple choice tests faster and more effective. The program provides the teacher with two primary tools: 

1.) Scan tests and save them as images
2.) Scan tests and grade them individually

This project is written in Python using OpenCV (Open Computer Vision) and EventBased Animation which is contained among project source files.


To run program, run webcam_scanner_final.py inside the Document Scanner directory. 

* Document Scanner - contains all files for project
* documentation - contains documentation about the project and its progression
* screenshots - visual display of what the program does behind the scenes
* video_demo - video demo of program in action
